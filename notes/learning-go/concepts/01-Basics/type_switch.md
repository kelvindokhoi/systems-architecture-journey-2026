# Type Switch in Go

Type switch is just a regular switch statement, but the cases are types instead of values.
Example:
```go
func printNumericValue(num interface{}) {
	switch v := num.(type) {
	case int:
		fmt.Printf("%T\n", v)
	case string:
		fmt.Printf("%T\n", v)
	default:
		fmt.Printf("%T\n", v)
	}
}

func main() {
	printNumericValue(1)
	// prints "int"

	printNumericValue("1")
	// prints "string"

	printNumericValue(struct{}{})
	// prints "struct {}"
}
```

Note: ```go
fmt.Printf("%T\n", v)
``` prints the type of the variable `v`.
